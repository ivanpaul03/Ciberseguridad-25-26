## Configuración

Para usar los modelos suele haber sistemas de pago, pero recomendamos
las opciones de **pesos abiertos** a través de **ollama cloud** y su
capa gratuita.

En el caso de Ollama, es compatible con **opencode**.

Del resto de herramientas, **cline** y **roo code** también lo permiten,
cada una con su configuración particular.

## Extensiones MCP

Para ampliar las acciones que pueden realizar los modelos se usa el
**Model Context Protocol (MCP)**.

Con MCP se pueden solicitar acciones como navegar por internet, crear
anotaciones en Notion, Jira, etc.

## Soporte MCP

La herramienta que permite habilitar servidores MCP de forma sencilla es
**Docker Desktop** en sus últimas versiones.
